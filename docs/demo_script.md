# Five-Minute Demo Script

1. Open `http://localhost:3000` and start on Overview.
2. Explain that VisionOps turns scattered YOLO experiment artifacts into a local review dashboard.
3. Point out the empty state, then click **Import Sample Data**.
4. On Overview, show experiment count, best mAP model, best FPS model, and failure case count.
5. Open Experiment Comparison and explain the multi-model accuracy/speed tradeoff.
6. Click one model row and open Model Run Detail to explain that run's metrics, curve, analysis, and related visual cases.
7. Open Failure Gallery and show how visual cases become reviewable evidence.
8. Open Inference Demo, upload `sample_data/inference_images/demo_upload.svg`, and run Demo Mode inference.
9. Open Demo Guide and summarize what Phase 1 proves and what later phases will add.
10. Close by saying Phase 1 proves the product loop, API boundary, database schema, frontend states, and Docker-ready project structure.
