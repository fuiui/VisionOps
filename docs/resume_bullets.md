# Resume Bullets

- Built VisionOps, a lightweight ML experiment management platform for computer vision research using FastAPI, React, SQLite, and Docker.
- Designed a reproducible Demo Mode workflow that lets reviewers run the project locally without private datasets or large model weights.
- Developed a dashboard for comparing computer vision model performance across precision, recall, mAP, FPS, and inference time.
- Implemented the backend API and database foundation for experiment tracking, visual failure analysis, and future YOLO inference.
